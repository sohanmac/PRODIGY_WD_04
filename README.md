# Social Media Platform (React + Node + MongoDB)

Backend: `server` — Express + Mongoose
Frontend: `client` — React (Vite)

Quick start:

Install backend deps:

```bash
cd server
npm install
```

Start backend (dev):

```bash
npm run dev
```

Install frontend deps and run:

```bash
cd client
npm install
npm run dev
```

Set environment variables using `.env.example`.
